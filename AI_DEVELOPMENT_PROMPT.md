# ТЕХНИЧЕСКОЕ ЗАДАНИЕ И МАСТЕР-ПРОМТ ДЛЯ РАЗРАБОТКИ ПЛАТФОРМЫ

Ты — Senior Full-Stack разработчик и Lead UI/UX дизайнер с глубоким опытом создания современных, высоконагруженных социальных приложений и систем с максимальным уровнем безопасности.

Твоя задача — разработать архитектуру, дизайн-систему и полный рабочий код (Web + Mobile Adaptive) для современной социальной платформы, объединяющей форум/обсуждения и сервис знакомств с защищенным мессенджером.

---

## 1. ТЕХНИЧЕСКИЙ СТЕК (Рекомендуемый для высокой скорости и визуального качества)
- **Фронтенд:** Next.js (React), TypeScript, Tailwind CSS.
- **Анимации и UI:** Framer Motion (для плавных и впечатляющих переходов), Lucide Icons, Glassmorphism (эффект матового стекла), акценты в стиле Neumorphism.
- **Бэкенд / База данных:** Supabase (PostgreSQL, Auth, Realtime WebSockets для мгновенных сообщений в чате).
- **Безопасность чатов:** End-to-End Encryption (E2EE) на базе Web Crypto API. Генерация публичных/приватных ключей происходит на клиенте, сервер хранит исключительно зашифрованные сообщения.

---

## 2. ДИЗАЙН И UI/UX (Требование: "Максимально крутой и современный визуал")
- **Цветовая палитра:** Темная тема (Dark Mode) по умолчанию с глубокими неоновыми или градиентными акцентами (например, глубокий космический синий + кибер-фиолетовый или яркий изумрудный).
- **Анимации (Framer Motion):** Плавное появление карточек при скролле (fade-in/slide-up), микроинтеракции при клике на лайки/кнопки, плавная анимация открытия диалогов и переходов между страницами.
- **Мобильная адаптивность:** На смартфонах интерфейс превращается в удобное приложение (с нижней навигационной панелью — Bottom Navigation Bar), на ПК — полноценный, просторный дашборд.
- **Индикация загрузки:** Использование Skeleton Loading (эффект мерцающих блоков вместо пустых экранов).

## 3. ФУНКЦИОНАЛЬНЫЕ МОДУЛИ СИСТЕМЫ

### Модуль А: Авторизация и Профиль
- Регистрация/Вход (Email, Google, Telegram OAuth).
- Профиль пользователя: аватар, статус, био, интересы/теги, город, возраст, пол.

### Модуль Б: Главная лента (Обсуждения и Темы)
- Лента с интерактивными карточками тем для обсуждения.
- **Жесткое ограничение:** Каждый пользователь может создать только 1 активную тему (для защиты от спама и повышения ценности обсуждений). При создании новой темы старая отправляется в архив или удаляется.
- **Алгоритм сортировки:** Вкладка «Новые» и вкладка «Популярные» (вычисляется динамически по количеству лайков, комментариев и просмотров).
- **Поиск и фильтрация:** Живой поиск (Live Search) по заголовкам и тегам в темах.
- **Внутри темы:** Древовидные комментарии, возможность ставить быстрые реакции (эмодзи).

### Модуль В: Раздел "Знакомства / Люди"
- Отдельная страница/вкладка со списком всех зарегистрированных пользователей в виде стильной сетки карточек (Grid).
- **Система фильтров:** По возрасту (минимальный/максимальный), полу, городу, интересам/тегам, статусу «в сети».
- Поиск по имени, никнейму или ключевым словам из био.
- При клике на карточку открывается детальный профиль пользователя с кнопкой «Написать сообщение».

### Модуль Г: Защищенный мессенджер (Личные чаты)
- Список диалогов с индикаторами онлайна и счетчиком непрочитанных сообщений.
- Экран чата: мгновенная отправка и получение сообщений (Realtime WebSockets через Supabase). Отображение статуса "печатает...".
- **Шифрование (E2EE):** Сообщения шифруются на устройстве отправителя (через публичный ключ получателя) и расшифровываются только на устройстве получателя (его приватным ключом). В базе данных сообщения хранятся исключительно в виде зашифрованного текста (cipher text). 
- Визуальный маркер в интерфейсе: "🔒 Чат защищен сквозным шифрованием".
- Надежное сохранение и синхронизация зашифрованной истории чатов.
