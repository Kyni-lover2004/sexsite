# 🌌 Nebula — Discussion & Dating Web Messenger (E2EE)

Nebula — это современная, конфиденциальная социальная платформа со сквозным шифрованием (E2EE) сообщений, интерактивным форумом (обсуждения), сервисом знакомств (поиск людей) и функционалом Premium-подписок.

Интерфейс выполнен в премиальном темном космическом стиле с золотыми неоновыми элементами, эффектом матового стекла (glassmorphic UI) и плавными микро-анимациями.

---

## ✨ Основные возможности

### 🔐 1. Сверхзащищённый чат (End-to-End Encryption)
- Сообщения шифруются на клиенте перед отправкой с использованием алгоритма **ECDH-P256** и **AES-GCM**.
- Сервер видит только нечитаемый шифротекст. Приватный ключ никогда не покидает браузер (хранится локально в безопасном хранилище IndexedDB).
- Поддержка отправки зашифрованных медиафайлов (фото/видео).

### 💬 2. Обсуждения (Форум)
- Возможность создавать темы с прикреплением медиафайлов.
- Дерево комментариев (ответы на комментарии) с автоматическим пересчетом количества лайков и просмотров.
- **Ограничение:** 1 активная тема на пользователя (старые автоматически отправляются в архив).

### 👥 3. Знакомства (Люди)
- Интерактивный поиск с фильтрацией:
  - По полу (я ищу: девушку / мужчину / всех)
  - По городу, возрасту и интересам
  - По активности (только онлайн / готовы пообщаться прямо сейчас)
- Пульсирующий индикатор **«Общаюсь сейчас»** для пользователей, готовых к диалогу.

### 👑 4. Nebula Premium
Отдельный красивый раздел с возможностью покупки тарифных планов (Неделя / Месяц / Год) и описанием фич:
- **Невидимка:** Скрытие вашего онлайн-статуса от других.
- **Безлимит:** Снятие лимита на знакомства (для бесплатных аккаунтов лимит составляет **2 новых контакта в день**).
- **Приоритет в поиске:** Карточка отображается выше других в списке людей.
- **Особый дизайн:** Золотой значок **PRO** и золотая рамка вокруг аватара.

### 🛡️ 5. Панель управления (Админка)
Защищенный раздел `/admin` (доступный только пользователям с ролью `admin`):
- Панель статистики (всего пользователей, топиков, забаненных).
- Управление пользователями: список с поиском, выдача банов/разбанов, роли.
- Управление контентом: поиск тем, удаление нарушающих правила постов с подтверждением.

## 🛠️ Технологический стек

- **Frontend:** Next.js 14 (App Router), React 18, Tailwind CSS, Framer Motion (анимации), Lucide React (иконки).
- **Backend/Database:** Supabase (Auth, PostgreSQL Database, Realtime, Storage).
- **Шифрование:** Web Crypto API (клиентское шифрование ключей).

## 🚀 Быстрый запуск

### 1. Установка зависимостей
```bash
npm install
```

### 2. Настройка окружения
Создайте файл `.env.local` в корне проекта и укажите ваши ключи Supabase:
```env
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
```

### 3. Запуск локального сервера разработки
```bash
npm run dev
```
Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## 💾 Применение схемы БД (Supabase)

Чтобы все новые фичи (Premium, админка, комментарии) работали стабильно, импортируйте полный скрипт `supabase/schema.sql` в **SQL Editor** в вашей панели управления Supabase и нажмите **Run**.

Это создаст нужные таблицы, индексы, триггеры счетчиков и RLS политики:
- Добавит колонки `role`, `is_banned`, `premium_until` в `profiles`.
- Добавит колонку `initiator_id` в `conversations`.
- Добавит колонку `media` в `topics` и `metadata` в `messages`.
- Настроит триггеры с параметром `SECURITY DEFINER` (для фикса ошибок RLS с комментариями/лайками).
- Создаст публичные бакеты `avatars`, `chat-images`, `profile-photos`, `topic-media`.

### Назначение себя администратором (SQL):
```sql
UPDATE public.profiles SET role = 'admin' WHERE username = 'ваш_username';
```

## 🛡️ Безопасность и RLS политики

- Все таблицы защищены с помощью **Row Level Security (RLS)**.
- Доступ к чатам разрешен строго двум участникам диалога.
- Запись в публичные папки хранилища (Storage) разрешена только авторизованным пользователям с проверкой `auth.uid() IS NOT NULL`, а чтение медиа доступно всем по прямым ссылкам.
