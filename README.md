# Desire Privé — Discussion & Dating Web Messenger (E2EE)

Desire Privé — это современная, конфиденциальная социальная платформа со сквозным шифрованием (E2EE) сообщений, интерактивным форумом (обсуждения), сервисом знакомств (поиск людей) и функционалом Premium-подписок.

Интерфейс выполнен в премиальном темном космическом стиле с золотыми неоновыми элементами, эффектом матового стекла (glassmorphic UI) и плавными микро-анимациями.

---

## ✨ Основные возможности

### 🔐 1. Сверхзащищённый чат (End-to-End Encryption)
- Сообщения шифруются на клиенте перед отправкой с использованием алгоритма **ECDH-P256** и **AES-GCM**.
- Сервер видит только нечитаемый шифротекст. Приватный ключ никогда не покидает браузер (хранится локально в безопасном хранилище IndexedDB).
- Поддержка отправки зашифрованных медиафайлов (фото/видео).

### 💬 2. Обсуждения (Форум)
- Возможность создавать темы с прикреплением медиафайлов.
- Дерево комментариев (ответы на комментарии) с автоматическим пересчетом количества лайков и просмотров.
- **Ограничение постов:** 1 активная тема на пользователя (предыдущие автоматически отправляются в архив). **Для администраторов это ограничение снято** — они могут вести несколько активных дискуссий одновременно.
- **Типы публикаций (для админов):** возможность помечать темы как *Обсуждение*, *Новость* или *Реклама / Промо*. Каждая публикация имеет уникальный дизайн:
  - *Новости* подсвечиваются мягким розовато-красным неоном со значком `✨ Новость`.
  - *Реклама / Промо* подсвечивается янтарно-золотым свечением со значком `👑 Промо`.
- **Уникальные просмотры:** система засчитывает просмотр темы только один раз от одного авторизованного пользователя с использованием связующей таблицы `topic_views` в БД, что защищает от накруток.

### 👥 3. Знакомства (Люди)
- Интерактивный поиск с фильтрацией:
  - По полу (я ищу: девушку / мужчину / всех)
  - По городу, возрасту и интересам
  - По активности (только онлайн / готовы пообщаться прямо сейчас)
- Пульсирующий индикатор **«Общаюсь сейчас»** для пользователей, готовых к диалогу.

### 👑 4. Desire Privé Premium
Отдельный красивый раздел с возможностью покупки тарифных планов (Неделя / Месяц / Год) и описанием фич:
- **Невидимка:** Скрытие вашего онлайн-статуса от других.
- **Безлимит:** Снятие лимита на знакомства (для бесплатных аккаунтов лимит составляет **2 новых контакта в день**).
- **Приоритет в поиске:** Карточка отображается выше других в списке людей.
- **Особый дизайн:** Золотой значок **PRO** и золотая рамка вокруг аватара.

### 🛡️ 5. Панель управления и Модерация (Админка)
Защищенный раздел `/admin` (доступный только пользователям с ролью `admin`) и расширенные возможности модерации на сайте:
- Панель статистики (всего пользователей, премиум, забаненных, тикетов).
- **Управление пользователями:** список с поиском, выдача банов/разбанов, роли.
- **Управление контентом:** поиск тем, удаление нарушающих правила топиков.
- **Модерация комментариев:** администраторы могут удалять любые некорректные сообщения под темами.
- **Модерация фотографий:** администраторы могут удалять любые фотографии из галереи пользователей прямо на страницах их профилей.
- **Модерация чатов:** специальная вкладка в админ-панели для просмотра списка всех активных DMs-переписок в системе и их полного удаления.

### 🧹 6. Автоматическая очистка базы данных
- Настроен планировщик `pg_cron` в PostgreSQL. 
- Каждый день в 3:00 ночи система автоматически удаляет пользователей, у которых **нет реальной активности более 30 дней** (поле `last_active_at`; администраторы исключены).
- Режим **невидимки** скрывает публичный `last_seen` / «в сети», но **не останавливает** `last_active_at` — пока человек пользуется сайтом, аккаунт не считается брошенным.
- Благодаря настроенным каскадным связям (`ON DELETE CASCADE`), вместе с неактивным аккаунтом без ошибок удаляются все его обсуждения, сообщения в чатах и комментарии.
- SQL: `supabase/patch_last_active_retention.sql` (колонка + `heartbeat` + `delete_inactive_profiles`).

## 🛠️ Технологический стек

- **Frontend:** Next.js 14 (App Router), React 18, Tailwind CSS, Framer Motion (анимации), Lucide React (иконки).
- **Backend/Database:** Supabase (Auth, PostgreSQL Database, Realtime, Storage).
- **Шифрование:** Web Crypto API (клиентское шифрование ключей).

## 🚀 Быстрый запуск

### 1. Установка зависимостей
```bash
npm install
```

### 2. Настройка окружения
Создайте файл `.env.local` в корне проекта и укажите ваши ключи Supabase:
```env
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
```

### 3. Запуск локального сервера разработки
```bash
npm run dev
```
Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## 💾 Применение схемы БД (Supabase)

Чтобы все фичи работали стабильно, импортируйте полный скрипт `supabase/schema.sql` в **SQL Editor** в вашей панели управления Supabase и нажмите **Run**.

## 🛡️ Безопасность и RLS политики

- Все таблицы защищены с помощью **Row Level Security (RLS)**.
- Доступ к чатам разрешен строго двум участникам диалога, либо администраторам при модерации/удалении чатов.
- Запись в публичные папки хранилища (Storage) разрешена только авторизованным пользователям с проверкой `auth.uid() IS NOT NULL`, а удаление медиа — владельцам файлов и администраторам сайта.
